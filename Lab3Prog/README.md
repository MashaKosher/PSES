## Lab3Prog — визуализация RGB по UART от STM32

Приложение на Python, которое читает по UART цветовые кадры от платы NUCLEO (формат `0xFF R G B`) и показывает тот же цвет в окне.

### Установка зависимостей

```bash
cd PSES/Lab3Prog
pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

Далее:

- выбрать (или оставить найденный) COM‑порт ST‑LINK VCP (например, `COM3`);
- нажать **Connect**;
- крутить потенциометр на плате — цвет окна должен меняться так же, как рассчитывается на STM32.
